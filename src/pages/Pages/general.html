Thanks for installing Tab Group Helper

Instructions

Type
  • When selecting URL, it'll read the URLs of the tabs and group against that
  • When selecting Title, it'll read the title attribute of the tabs and group against that.
Contains
  • Write what you want to match against. If you picked "title" for Type, and you want to match YouTube tabs about cats, you might put "cats". If you picked "URL" for Type, and you want to match all Youtube tabs, you might put "youtube"
Case Sensitive
  • URLs and Titles can be tricky to match against, by default this option is off to ensure the best user experience. If you do need stricter rules, you can enable this
Regex
  • You can enable this and in the "Contains" input field write regular expressions. For example, if you want to group social media tabs together you might enable this and put "facebook|twitter|instagram|tiktok" in the "Contains" field
Group Name
  • The name of the tab in the browser
Color
  • The color of the tab in the browser
Auto-Group
  • "Created Tab" will group tabs when they are created, duplicated
  • "Updated Tab" will group tabs when they are updated, by title or URL. This would run on page changes for example
Apply
  • By default enabled, this turns on/off this rule
